Foundation Overview

Newtype 디자인 시스템의 파운데이션은 시각적 언어의 기초를 정의합니다.
우리는 아래의 원칙을 통해 일관되고 효율적인 사용자 경험을 만듭니다.

Principles

Newtype 서비스를 관통하는 3가지 핵심 디자인 가치입니다.

Clear

정보는 명확해야 합니다. 불필요한 장식을 배제하고 사용자가 핵심에 집중할 수 있도록 돕습니다.

Consistent

일관성은 신뢰를 만듭니다. 예측 가능한 패턴과 스타일을 사용하여 학습 비용을 줄입니다.

Efficient

효율은 속도입니다. 재사용 가능한 컴포넌트와 토큰을 통해 디자인과 개발 속도를 높입니다.

System Elements

디자인 시스템을 구성하는 가장 기초적인 시각적 요소들입니다.

Color

Typography

Layout

Iconography
