Layout

반응형 그리드 시스템은 다양한 디바이스 환경에서 일관된 배치와 비율을 제공합니다.
12컬럼 시스템을 기본으로 하며, 4px 간격 규칙을 따릅니다.

Breakpoints

디바이스 해상도에 따른 레이아웃 변경 기준점입니다.

Device Range Container Width Token
Mobile 0 ~ 767px 100% (Fluid) @media (max-width: 768px)
Tablet 768px ~ 1199px 720px ~ 960px @media (min-width: 768px)
Desktop 1200px ~ 1200px (Fixed) @media (min-width: 1200px)

Grid System

화면을 세로로 나누는 컬럼(Column) 구조입니다.

Columns Gutter (간격) Margin (외부 여백)
12 (Desktop) 24px Auto (Centered)
4 (Mobile) 16px 20px
