Color

컬러는 브랜드의 아이덴티티를 표현하고 정보의 위계와 상태를 전달하는 핵심 요소입니다.
Newtype 디자인 시스템은 명확한 목적을 가진 제한된 컬러 팔레트를 사용합니다.

Primary Brand

브랜드의 주요 색상으로 로고, 버튼, 강조 텍스트 등에 사용됩니다.

Black#000000--primary
Charcoal#333333--primary-dark

Neutrals

텍스트, 배경, 테두리 등 UI 전반의 구조를 잡는 데 사용되는 그레이스케일입니다.

Gray 900#111111Title Text
Gray 600#666666Body Text
Gray 400#999999Disabled
Gray 200#E5E5E5Border
Gray 100#F7F8FABackground

Semantic (Functional)

사용자에게 상태 피드백(성공, 에러, 경고)을 전달할 때 사용하는 기능적 컬러입니다.

Error / Danger#E02020--system-error
Success#06C755--system-success
Information#007AFF--system-info
