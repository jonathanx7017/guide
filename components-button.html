Button

버튼은 사용자가 클릭이나 탭을 통해 작업을 수행하게 하는 주요 인터랙션 요소입니다.
중요도에 따라 Primary, Secondary, Tertiary 스타일을 구분하여 사용합니다.

Types

기능의 우선순위에 따라 버튼 스타일을 구분합니다.

Primary

화면에서 가장 중요한 단 하나의 주요 액션에 사용합니다. (예: 저장, 구매, 로그인)

Secondary

주요 액션을 보조하거나, 대안적인 액션에 사용합니다. (예: 취소, 뒤로가기)

Tertiary (Ghost)

배경 없이 텍스트로만 구성되며, 중요도가 낮은 액션에 사용합니다.

States

State Visual Description
Enabled 기본 상태입니다.
Hover 마우스 커서가 올라갔을 때 (배경색 어둡게, Y축 이동)
Pressed 클릭하는 순간 (Scale 0.96 축소)
Disabled 비활성화 상태 (Opacity 처리, 클릭 불가)

Usage Guidelines

DO

화면 당 하나의 Primary 버튼만 사용하여 사용자의 주요 행동을 유도하세요.

DON'T

Primary 버튼을 남발하면 사용자가 무엇을 눌러야 할지 혼란스러워합니다.
