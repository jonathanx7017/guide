Accessibility

우리는 모든 사용자가 신체적, 환경적 제약 없이 제품을 사용할 수 있도록
WCAG 2.1 (AA Level) 표준을 준수합니다.

Core Principles

Perceivable

정보와 UI 요소는 사용자가 인지할 수 있는 방식으로 표시되어야 합니다. (대체 텍스트, 명도 대비)

Operable

모든 기능은 키보드만으로도 조작 가능해야 하며, 발작을 유발하는 요소를 피해야 합니다.

Robust

보조 기술(스크린 리더 등)을 포함한 다양한 사용자 에이전트가 콘텐츠를 안정적으로 해석해야 합니다.

Color Contrast

텍스트와 배경 간의 명도 대비는 최소 4.5:1을 유지해야 합니다.

Readable Text Ratio 14.3:1 (Pass)
DO

충분한 대비를 가진 색상 조합을 사용하여 저시력자도 쉽게 읽을 수 있게 합니다.

Low Contrast Ratio 1.8:1 (Fail)
DON'T

장식적인 요소가 아니라면, 텍스트에 연한 회색을 사용하지 마세요.
